37  /**
38  * Wrapper class for KnoRBA 3-state `truth` type. A value of type `truth`
39  * can be either `T` (for true), `F` (for false), or `X` (for unknown).
40  * These values are represented by `knorba::type::T`, `knorba::type::F`,
41  * and `knorba::type::X`, respectively. The scalar type associated with this
42  * wrapper class is `knorba::type::k_truth_t`.
43  *
44  * KnoRBA employes 3-state logic rather than the common Boolean logic mostly
45  * because it simplifies implementation of KnoIL interpreter and makes it
46  * easier for a group of agents to engage in collaborative descision making.
47  *
48  * @headerfile KTruth.h <knorba/type/KTruth.h>
49  */
